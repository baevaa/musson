Файлы можно корректировать в любом редакторе кода, 
но корректировки style.css и app.js требуют применения сборщика для преобразования файлов в сжатые версии.

Папка css содержит описание стилей блоков - файл style.css 
	style.min.css - сжатая версия

Папка fonts содержит шрифты используемые на сайте

Папка img - изображения

Папка js содержит описание функций при клике на элементы - файл app.js 
	app.min.js - сжатая версия
Папка pdf-link содержит файл для скачивания презентации

Файл favicon.ico - иконка сайта

Для редактирования текста и картинок открыть следующие файлы:

	index.html- русская версия сайта в собранном виде;
	english.html- английская версия сайта в собранном виде.
